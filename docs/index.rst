Welcome to pytest-containers
============================

A pytest plugin for docker and kubernetes fixtures.

Installation
------------

The latest stable version `is available on PyPI <https://pypi.python.org/pypi/pytest_containers/>`_. Either add ``pytest_containers`` to your ``requirements.txt`` file or install with pip::

    pip install pytest_containers
